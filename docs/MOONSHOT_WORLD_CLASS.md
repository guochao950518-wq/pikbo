## 2. 流量暴涨怎么来（谷歌 + 社群，不是玄学）

### 2.1 谷歌（哥飞路径 × 潮玩垂直）

```text
意图词（example）
  "blind box unboxing video from photo"
  "360 product video of figure no turntable"
  "etsy listing video action figure"
  "whatnot toy spin video maker"
        ↓
一页 = 搜索语言 H1 + 可玩工具（进 Create 带 recipe）+ 示例 + FAQ
        ↓
周更 3–5 页高质量，不程序化垃圾海
        ↓
GSC 收录 → 长尾累加 → 首页品牌词
```

**不做：** 1000 页薄 AI 文。谷歌 2024–26 杀的就是这个。  
**做：** 50 个 **真能完成工作** 的意图页 + 真片。

### 2.2 非谷歌（往往更快起量）

| 渠道 | 玩法 |
|------|------|
| TikTok / IG Reels | 「一张手办照 → 5 秒」前后对比；水印 pikbo.ai |
| Reddit / Discord 潮玩 | 规则内分享；不 spam |
| Etsy/Whatnot 卖家社群 | Seller OS 案例：listing 转化前后 |
| 创作者 affiliate | Shop 订阅分成（以后） |

**病毒核：** 输出本身就是广告（玩具动起来 = 停留）。
