## 2. 核心结论（给全员）

1. **HF 不是图生视频页**，是 **发现 + 配方 + 多模型 + 商品/影棚 + 资产** 的 OS。  
2. **Yiha** 是 **工具货架 + 模型超市 + 透明积分**。  
3. 两边都验证 **App 闭环**；Pikbo 只吃 **潮玩图→可卖片** 这一刀要更深。  
4. 最该抄的交互：**结果优先、一键完整配方、Create 四段、商品三步、积分可见**。  
5. 最不该抄：**空 Cinema、假模型墙、假社区**。  
6. 市场：HF run rate 远超日入 6 万；6 万是垂直份额不是幻想。
