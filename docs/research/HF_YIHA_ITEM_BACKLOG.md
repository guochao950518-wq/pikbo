## 0. 为什么「还差很远」（第一性原理）

他们赢的不是菜单数量，是这五件物理感：

| # | 他们有的感觉 | 本质 | Pikbo 合法对标 |
|---|--------------|------|----------------|
| 1 | **满屏能动的结果** | 媒体驱动进站 | Lab 真片墙 + 独立 proof |
| 2 | **一点就进工作台** | 深链 = 完整配方 | `?effect/?job/?mode` |
| 3 | **Flow / Apps 矩阵** | 创建入口货架 | `/flow` + `/modules` |
| 4 | **生产台闭环** | 上传→生成→资产→再 remix | Create + Library + versions |
| 5 | **规模与模型** | 多 provider + UGC | **不做假**；有 key 再上 |

差距大的不是「没有 Cinema 菜单」，而是 **媒体密度 + 入口矩阵 + 结果闭环** 还不够像「创作 OS」。
