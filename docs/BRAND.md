# Pikbo brand & product rules

## Positioning
**Designer-toy AI video studio** — same *product class* as Higgsfield/Yiha (app shell, models, presets, credits), **not** a clone of their look or brand.

## Steal the system, not the skin
| From big AI video apps | Pikbo keeps |
|---|---|
| Left nav app chrome | ✓ |
| Generate workspace | ✓ |
| Models / Apps / Cinema / Library | ✓ |
| Credits + plans | ✓ |
| Seedance (ByteDance) as engine | ✓ |
| Their lime skin, logos, copy, posters | ✗ |

## Our style (own it)
- **Palette:** candy pink `#ff4d8d` + violet `#a855f7` (not competitor lime-as-identity)
- **Accent lime** only as a small “live/tech” chip, not the whole brand
- **Personality:** toys, shelves, unboxing, sellers — warm, playful, premium collectible
- **Voice:** clear tool SaaS, not generic “AI film empire”
- **Hero product:** figure / blind box photo → clip, not face-swap celebrity memes

## Tagline options
- Primary: *Bring your designer toys to life*
- Product: *AI video studio for figures & collectibles*

## When building UI
1. Structure like a full creative suite (so it feels pro).
2. Style like Pikbo (so it feels ours).
3. Default niche copy toward toys; suite pages can stay slightly general.
