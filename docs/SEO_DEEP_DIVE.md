## 1. 搜索引擎如何工作  
https://ahrefs.com/blog/zh/how-do-search-engines-work/  **【全文】**

### 核心模型
1. **索引** = 网页数据库  
2. **算法** = 从索引里匹配并排序

### 建索引四步
URL 发现（反链 / sitemap / 手动提交）→ **抓取**（Googlebot）→ **处理与渲染** → **写入索引**

### 排名关键因素（已知公开部分）
- **反向链接**（量 + 质；引用域名与流量正相关）  
- **相关性**（关键词 + 互动数据）  
- **内容新鲜度**（查询依赖：新闻强，解魔方弱）  
- **页面速度**（偏惩罚慢站）  
- **移动友好**（移动优先索引）

### 个性化
位置、语言、搜索历史 → 同一词不同人看到不同 SERP

### 新手要记住的一句话
**没进索引 = 不存在。** 进索引 ≠ 有排名；排名靠相关 + 信任（链）+ 体验。

## 2. SEO 基础：新手成功指南  
https://ahrefs.com/blog/zh/seo-basics/  **【全文】**

### SEO 定义（2025 版）
优化站与内容，提升在**搜索引擎 + AI 助手**中的可见度（点击、摘要、被提及）。

### 五步闭环
1. 关键词研究  
2. 内容创作  
3. 页面 SEO + 结构  
4. 链接与品牌提及  
5. 技术 SEO  

### 成功设置清单
- 域名：短、好记；优先 .com  
- 平台：WordPress / 托管站均可；Pikbo 用 Next 也 OK  
- 主机：安全 HTTPS、近用户、可配 robots/sitemap  
- UX：移动、字号、少弹窗、快  
- **逻辑站点结构 + 内链**（思维导图 → 导航/面包屑）  
- URL：描述性 slug（/seo-basics/ 优于 ?p=123）  
- 本地业务才做 Local SEO  

### 收录
找/建 sitemap → GSC 提交  

### 衡量
自然流量、展示量、关键词排名、AI 可见度、声音份额

## 3. 关键词研究  
https://ahrefs.com/blog/zh/keyword-research/  **【全文】**

### 为什么致命
**90%+ 页面零 Google 流量**——常见原因：写了没人搜的东西。

### 找词四源
1. 种子词 + 免费生成器 / ChatGPT  brainstorm  
2. 自己站已排名词（GSC + Ahrefs Free）  
3. 竞品 Top pages / Content Gap  
4. 论坛/Reddit/客户原话  

### 类型
词匹配 / 短语匹配 / 相关词 / **父主题聚类** / 意图（信息/商业/交易/导航）

### 六大指标
| 指标 | 用法 |
|------|------|
| 搜索量 | 年均值；≠ 你能拿到的流量 |
| **流量潜力** | 看排名第一页总流量，比单量更真 |
| KD | 进前 10 约需多少引用域；别只筛 0–10 |
| CPC | 商业价值代理；广告主愿付多少 |
| 增长/趋势 | 抓上升词；避开假平均 |
| 商业潜力 0–3 | 能否自然推你的产品 |

### 优先级哲学
「看情况」：新站 → 低难长尾；要转化 → 高商业潜力；短中长期目标要同时有。

## 4. SEO Content（仅英文）  
https://ahrefs.com/blog/seo-content/  **【全文】**

### 什么是 SEO 内容
为搜索者**和**搜索引擎设计的内容：博客、落地页、**交互工具**都可以。

### 创作流程（必背）
1. **选 proven topic**（流量潜力 + 商业潜力 + 可排名）  
2. **分析搜索意图** → 3C：Content type / format / angle（看 SERP 长什么样）  
3. **检查 EEAT**（一手经验、专家、证据）  
4. **覆盖完整**（穷尽子话题，不等于又臭又长）  
5. **做出独特性**（信息增量：数据、实验、观点、调研）  
6. **Sticky**（好读、视觉、受众语言）  
7. **Evergreen 可更新**（会烂的主题要维护）

### 反抄袭警告
只抄 SERP 前列 = copycat → 没人链你 → 难排名。  
**Information Gain** 是核心竞争力。

## 6. 链接建设  
https://ahrefs.com/blog/zh/link-building/  **【全文】**

### 本质
链接 ≈ 投票；质量 > 数量。

### 四类手法
1. **添加**（目录/社媒/论坛）→ 基础外链，权重低但新站可做  
2. **请求**（外展：客座、摩天楼、失效链、资源页、HARO…）  
3. **购买** → Google 视为操纵，本指南不教  
4. **赢得** → 可链接资产 + 推广  

### 好链接五维
权威 / 相关 / 锚文本（别过度控制）/ 位置 / 链到哪（首页 vs 资产 vs 赚钱页）

### 最佳策略
抄竞品外链、针对 SERP 外展、可链接资产、内容推广、客座  

### 比喻
人无聊难交友；站无聊难获链。

## 7. 技术 SEO  
https://ahrefs.com/blog/zh/technical-seo/  **【全文】**

### 定义
让搜索引擎（和 AI）**发现、抓取、理解、索引**你的页。

### 速胜优先级
1. 检查索引（noindex / 404 / 屏蔽）  
2. 301 救失效外链页  
3. 加内链  
4. Schema  

### AI 技术点
- 核心内容勿只藏 JS 后（LLM 常不渲染 JS）  
- 别默认 Cloudflare 拦 AI 爬虫（若要 GEO）  
- 幻觉 URL 做 301  

### 哲学
技术出故障时优先修；平时**内容+链接**往往 ROI 更高。
